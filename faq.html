常问问题搜索

项目状态
金盾老板总控台现在做什么？
只做老板管理入口：看状态、看告警、看下一步；不是自动交易系统。
刷新
页面会自动刷新吗？
会。页面每 60 秒重新读取 data/status.json，不需要手动 F5。
stale
stale 是什么意思？
状态文件太久没更新。页面会变黄色，提醒先查 GitHub Actions 自动同步有没有停。
running
黄色 running 是什么意思？
GitHub Actions 正在运行或排队，等流水线结束后页面会自动读新状态。
失败
红色失败是什么意思？
最近 workflow 或同步流水线失败，优先查 Actions 日志，不要继续加功能。
GitHub
GitHub Actions 自动同步怎么做？
主仓生成安全摘要 JSON，只把公开字段推送到老板总控台仓库的 data/status.json。
交易
现在能不能交易？
不能。公开网页不下单、不接 live、不改 lot / SL / TP / 策略参数。
安全
公开页会不会泄露东西？
不会放密钥、Token、账号密码、主仓原始日志或机器敏感信息，只放安全摘要。
版本
这是新版本吗？
不是乱开新版本。主线名称保持 V6.1 patch，只做合并补丁。
